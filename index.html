Features

Zoomable

OpenSeadragon image zooming using the IIIF image API

Embeddable

YouTube-style embedding with deep links to specific pages/zoom regions.

Translatable

Internationalised UI supporting English, Welsh, Swedish and more.

Configurable

The UV is highly configurable with an array of customisable settings.

Extensible

Supports 3D, audio, video, and pdf viewing experiences.

Searchable

Supports search and autocomplete service integration with overlaid search results.

Securable

The UV supports the IIIF Auth API, so your content can be password protected.
